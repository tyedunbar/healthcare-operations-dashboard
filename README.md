# Healthcare Operations Dashboard

## Overview
This project analyzes healthcare data to identify trends in patient volume, wait times, and departmental performance.

## Tools Used
- Power BI
- Excel

## Key Insights
- Identified peak patient hours
- Highlighted departments with longer wait times
- Provided recommendations to improve efficiency

## Outcome
This project demonstrates how data visualization can support better decision-making in healthcare operations.
